# QPushButton颜色、格式设置

1.在属性栏找到stylesheet，点击旁边的省略号

![](https://mawenshui.oss-cn-beijing.aliyuncs.com/img_Markdown/202203311424714.jpg)

2.输入以下代码：

QPushButton { background-color: rgb(255, 132, 139); border-radius: 3 px; color: rgb(255, 255, 255); } QPushButton:hover { background-color: rgb(245, 69, 234); }

其中hover属性设置的是鼠标放上去变化的颜色。rgb里面是颜色的代码，可自己查询

![](https://mawenshui.oss-cn-beijing.aliyuncs.com/img_Markdown/202203311424854.jpg)
